# 平台

# 说明

平台，这里指运行环境，包括系统，浏览器，域名等。此模块包含了一些有关平台的方法，导出模块名为platform

  • 导入模块
import { platform } from 'gx-instant'

# 方法

# getBrowser

说明：获取浏览器类型

  • 参数：无
  • 返回：string 浏览器类型，如chrome

# getOS

说明：获取系统类型

  • 参数：无
  • 返回：string 系统类型，windows

# isPc

说明：判断是否为pc端

  • 参数：无
  • 返回：boolean 判断结果

# isMobile

说明：判断是否为移动端

  • 参数：无
  • 返回：boolean 判断结果

# getEnv

说明：获取当前环境信息

  • 参数：无
  • 返回：Object 包含浏览器，操作系统，域名地址信息的对象，格式： {browser,os,host}

# getHost

说明：获取当前域名地址

  • 参数
    • url: string url地址，可选，默认是浏览器地址
    • pre：boolean 是否带协议前缀，可选，默认false
  • 返回：string 域名信息
